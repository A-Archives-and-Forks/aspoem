# aspoem.com

现代化中国诗词学习网站，一个更加注重 **UI和阅读体验**的诗词网站。

## 基础技术栈

- prisma
- nextjs
- trpc
- tailwindcss
- shadcn/ui
- postgres + supabase
- twikoo
