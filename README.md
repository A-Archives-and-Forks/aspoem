## 技术栈

- prisma
- nextjs
- tailwindcss
- shadcn/ui
- postgres + supabase
- twikoo

## 本地部署

参考 `.env.example` 配置。

## 规范

### 标题

1. 删除标题中的量词，其一，一等，用正文中的第一句代替。

```
行路难其一 => 行路难·大道如青天
```

2. 删除标题中的描述，杂曲歌辞 等描述说明，只保留标题

3. 如果只有词牌名，用 “词牌名·第一句”

### 内容

1. 诗：采用居中对齐方式
2. 词：采用左对齐，开头缩进2个字符

### 样式

| 中文 | font-cursive |
| ---- | ------------ |
| 英文 | font-serif   |
| 数字 | font-mono    |
